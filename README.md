# Practicing Node

Tiny Express project for basic Node.js route practice and quick daily commits.

## Run

```bash
npm install
npm run dev
```

Or:

```bash
npm start
```

## Routes

- `GET /` simple text response
- `GET /about` simple JSON response
- `GET /api/tips` array JSON response
- `GET /api/user/:id` route param example
- `POST /api/echo` request body echo example
